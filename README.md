# CLUBBER 애플리케이션
Android Studio, Firebase를 활용한 대학생 동아리 관리, 신청 앱 입니다.

## 개발 동기
대학교 동아리에 대한 정보가 잘 정리되어 있지 않아, 원하는 동아리를 찾고 신청하기 어려워 해당 앱을 제작하였습니다. 

## 개발 목표
신청할 수 있는 대학교 동아리 리스트를 분류에 따라 정리하고, D-Day를 제공하여 정보 수집을 편리하게 한다.
